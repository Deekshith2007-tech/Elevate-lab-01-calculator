# Elevate-lab-01-calculator
To build a command line calculator in python to supports arithmetic operations.
In this Task i have used functions,loops,conditions,command line interface.
In this python program i have used user defined function def() and exception handling such as try except. 
It performs addition,subtraction,multiplication and division.and user function for each operation
It takes user input as input() and handles division by zero as ValueError and invalid inputs.
